Insure Predict Architecture

Overview of the machine learning prediction architecture.

Our insurance prediction system uses two separate machine learning models to analyze the profile details and infer premiums:

  • Discount Classifier (SVC): A Support Vector Classifier (SVC) trained to evaluate client details and classify whether the individual qualifies for dynamic wellness discount rates.
  • Expense Regressor (OLS): A Linear Regression model that takes client inputs along with the generated discount eligibility profile as inputs to predict the final annual premium.

By nesting the prediction flows, the pipeline mimics actual underwriting policies where qualifications are calculated first and then applied to pricing matrices.
